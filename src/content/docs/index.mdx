## 🚀 Sistema de Gestión de Transporte de Carga

<div class="text-lg text-zinc-400 mb-8">
  Este sitio aloja el **expediente técnico integral** desarrollado para <span class="text-emerald-400 font-semibold">Transportadora Colque Machicado S.R.L.</span> Su propósito es centralizar la información comercial, operativa y de facturación mediante un enfoque híbrido de ingeniería de software que combina el **Análisis Estructurado** para la persistencia y flujo de datos, con el **Diseño Orientado a Objetos** para la arquitectura lógica del sistema.
</div>

<div class="my-10">
<CardGrid>
  <Card title="📐 Diseño Estructurado" icon="setting">Modelado de procesos mediante Diagramas de Flujo de Datos (DFD), Diccionario de Datos y el diseño base de datos relacional (Entidad-Relación).</Card>
  <Card title="🧩 Orientado a Objetos" icon="seti:json">Representación del dominio del sistema mediante artefactos UML: Diagramas de Clases, Casos de Uso y Secuencia para un software escalable.</Card>
  <Card title="🚛 Control Operativo" icon="rocket">Arquitectura diseñada para la generación de guías, asignación eficiente de choferes y trazabilidad completa de viajes.</Card>
  <Card title="🧾 Facturación SIAT" icon="pencil">Integración modular orientada a servicios para la comunicación, emisión y validación de facturas con Impuestos Nacionales.</Card>
</CardGrid>
</div>

</ContainerSection>
